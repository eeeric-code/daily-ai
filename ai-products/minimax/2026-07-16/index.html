🎓 学术 Mozilla 旗下 0din 团队披露 Claude Code 漏洞:攻击链以一个普通 Markdown 安装说明文件为起点,内容说明如何安装 Axiom 监控工具,运行未初始化工具弹出错误提示,指示执行特定配置命令。Claude Code 出于「帮助用户」本能,自动遵循该指令,触发隐藏 Shell 脚本,脚本从攻击者控制的 DNS 文本记录中获取 Base64 编码的反向 Shell 命令,在静默状态下连回攻击者服务器。
