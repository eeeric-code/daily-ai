Anthropic Claude Science 科研 Agent 60+ 技能连接器 NVIDIA Evo 2 生命科学

🎓 学术7月2日 Anthropic 正式发布 Claude Science,这是一款面向科学家的 AI 工作台,将分散科研工具整合到统一环境,核心是一个协调代理,可调用 60+ 预置技能与连接器,覆盖基因组学、蛋白质组学、结构生物学与化学信息学;独立的"审核代理"实时检查引文与计算结果,标记并纠正错误;每张生成的数据图附带生成代码、环境信息与对话历史,支持数月后追溯任何结果;可访问 NVIDIA Evo 2、Boltz-2、OpenFold3 等生命科学模型,接入 UniProt、PDB、ChEMBL 等 60+ 科学数据库。

🏢 产业这是 Anthropic 冲刺 IPO 前争夺付费客户的关键布局,产品运行在实验室自有基础设施(macOS/Linux 本地或 SSH/HPC 远程),大型作业由代理规划后提交至实验室自有集群或按需计算账户,工作负载可从单 GPU 扩展至数百个,数据不出实验室,既解决了数据隐私也吃下 NVIDIA 生态协同;早期测试者 Manifold Bio(药物靶点筛选)、艾伦研究所神经科学(多代理文献综述)、UCSF 流行病学家(胶质瘤分析提速 10 倍)反馈积极。

关键数据:已在 macOS/Linux 对 Pro/Max/Team/Enterprise 开放 Beta;学术与非营利实验室享折扣席位;Anthropic 资助最多 50 个研究项目,每项最高 3 万美元 API 积分;基于现有 Claude 模型,非新模型。
