### 5.2 对每条 action 找真源（不限 site:，全网搜）：

```python
for action in actions:
    real_url = None
    # 优先用 action.card_title 作为核心关键词
    keywords = extract_keywords(action.card_title)  # 至少 3 个
    
    for attempt in range(max_attempts):  # max_attempts = 3
        # 1. web_search：全网搜（不限 site:）
        query = f"{keywords} {today}"  # 不加 site: 限定
        results = web_search(query, top_n=8)
        
        for r in results:
            # 2. 滤 fake-200 关键词
            if any(kw in r.snippet for kw in ["404","Not Found","Cloudflare","Access Denied"]):
                continue
            
            # 3. webfetch 验证 200
            try:
                resp = webfetch(r.url)
                if resp.status_code != 200:
                    continue
                body = resp.text
                if any(kw in body for kw in fake_200_keywords):
                    continue
                
                # 4. 主题词匹配（≥2 关键词）
                if sum(1 for kw in keywords if kw in r.url or kw in body[:1500]) >= 2:
                    real_url = r.url
                    break
            except:
                continue
        
        if real_url:
            break
        # Fallback: 用 alternative_domains 替换搜（仍不限 site:）
        for alt_domain in action.alternative_domains:
            new_query = f"{keywords} {today}"  # 不加 site:
            results = web_search(new_query, top_n=5)
            # ... 同样验证流程
    
    action.real_url = real_url  # None = 找不到
```

if (-not $homepageCardOk) {
    Write-Host "✗ 首页 minimax 综合卡片**未同步**,需立即 patch" 'WARN'
    # 自动 patch 部署版 repo/index.html
    $deployedIndex = "D:\ECNU\其他\每日AI推送\repo\index.html"

    $content = $content -replace '(<span class="model-name">minimax</span>[\s\S]*?<span>已收录 ?)\d+( ?期</span>)', "`$1$expectedMinimaxCount`$2"
    $content = $content -replace '(<span class="model-name">minimax</span>[\s\S]*?href="\./ai-daily/minimax/)\d{4}-\d{2}-\d{2}(/)"', "`$1$expectedDate`$2"

## 九、验证风控（v2.7 强化）

| 风险 | 缓解 |
|---|---|
| mavis agent 跳过 commit + push | 无硬超时，但 session lifecycle 自动 finish（兜底） |
| LLM 编假 URL | web_search 全网搜（不限 site:），避免漏转载 |

| 自动 push 错内容 | 每轮 commit 前重 audit，期望 url_fail=0 |
| 转载源漏报（v2.5/v2.6 bug） | v2.7 search_query 不限 site:，用 card_title 全网搜 |
